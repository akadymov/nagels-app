Game info

Nägels is card game for 2-6 players.

History of the game is not clear, but it seems like it comes from swiss traditions (you are welcome to reveal correct and full story of Nägels if you know it for a fact - in that case please use feedback form).

Being a hybrid between "Preferans" and "Hearts" (installed by default on some earlier Windows systems), Nägels develops strategy and logic skills. At the same time it gives an opportunity to play and have fun with friends. Rules can seem a bit complex, but when you memorize basic rules and deep into the game process you'll recognize how exciting it is.

Good luck!

Nägels Online dev team

Game rules

The goal is to score maximum points during whole game, that is decomposed to 10-20 hands.

Here are main rules:

  1. Number of players. There should be from 2 to 6 players to start game. Optimal is 4-5.
  2. Number of dealt cards in hand. Every hand number of dealt cards changes by 1 (exception - hand with 1 card is played two times in a row): first, number of cards is reduced to 1, then - increased to initial number.
  3. Trump. Trump is changed every hand in strict order:

    diamonds

    hearts

    clubs

    spades

    no trump

  4. Bets. Every hand players make bet (zero or more) in counterclockwise order. Starting player is changed every hand counterclockwise. Sum of bet sizes should NOT equal number of dealt cards per player. That’s why there is restriction for last betting player only (someone should stay unhappy).
  5. Putting cards in turn. After making all bets players start to put cards on table. No restriction in choosing card for first putting player, but every next player has to put card in suit of first card or trump suit. There are two exceptions: player can put any card if he/she doesn’t have suited card and can NOT put trump lower than already put trump when first card is NOT trump (you can not leak trumps). When every player puts one card (this is one turn of the hand) cards are took by player with highest suited card or trump. Keep in mind that Jack of trumps is highest card, and second is Nine! This rule applies only for trump suit.
  6. Hand scores. After all turns are made took number of turns are calculated. Every player gets points equally to number of took turns + 10 points IF number of took hands equals to made bet.
  7. Game scores. In the end of game (after all hands) player is defined based on cumulative number of scored points.

Game start

After at least one player joins lobby room its host can start the game. Ready switch is introduced just for players communication.

When starting the game host can configure following options

  • Auto deal cards - if option is checked, cards are dealt automatically on every hand start, otherwise host can choose when to press "Deal" button manually
  • Single card hands - if option is checked two hands with one card per player is played in middle of the game, otherwise two cards are dealt in these hands
  • Rating game - if option is checked game results will be written to players profiles and overall leaderboards (no data is written if all hands not played)

Timebank option: under construction

On game start host should initiate automatic defining players' positions in the game. Game starting player has a dealer button on the left. Starting player is changed every hand counterclockwise.

Betting

Then first hand of the game starts with card dealing. First round of hand is making bets. Every player has to make a bet at his/her order taking into account dealt cards (including cards number), already made bets and current trump.

Following modal displayed when it comes your turn to bet.

Player cannot make a bet that is higher than number of cards on hand.

Last betting player cannot make a bet which, summed with already made bets, equals to the sum of cards on hand.

Putting cards

Then every player puts card in his/her turn. Player who put highest card takes the turn and starts next one.

Following action message displayed when it comes your turn to put card.

On your turn select any card from your hand and confirm choise by clicking/tapping it second time.

Following message displayed in case putting selected card in this turn is prohibited.

After putting last card in hand new hand is dealt.

Scores and standings

You can check current standings by clicking "Scores" at any point of game.

Finishing game

After finishing the game winner username is displayed.

Host can finish the game at any point by clicking/tapping "Finish".

Player can exit game and room at any point by clicking/tapping "Exit".

Host can instantly start new game in the room with same configuration by clicking/tapping "Start new game".

Leaderboards and stats

You can view overall players statistics in Leaderboard section.

Also individual user stat are available in player profile which is available by clicking/tapping on player's username on game table
